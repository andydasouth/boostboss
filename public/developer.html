Integration Guide

1
Add MCP Server

Configure your MCP server endpoint in your AI application.

{
  "mcp_server": {
    "type": "sse",
    "url": "https://api.boostboss.ai/mcp",
    "api_key": "bb_dev_sk_loading"
  }
}

2
Add SDK

Include the Boost Boss SDK in your application.

<script src="https://cdn.boostboss.ai/sdk/v1/boost-boss.js"></script>
<script>
  BoostBoss.init({
    apiKey: 'bb_dev_sk_loading',
    appId: 'app_loading'
  });
</script>
